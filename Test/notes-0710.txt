ch 5. induction proof & recursion
  - mathematical induction 
  - strong induction 


  mathematical induction
    - used to prove P(n) is true for all positive integers n 
    - steps:
      1. show P(1) is true 
      2. show P(k) -> P(k + 1) is true for all positive integers k

  strong induction
    - to prove P(n) is true for all positive integers n show:
      1. basis step: verify P(1) is true 
      2. inductive step: show [P(1) && P(2) && ... && P(k)] -> P(k + 1)
                         holds for all positive integers k
